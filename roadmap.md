# 🗺️ Project Roadmap – Ultimate Backport Mod
This roadmap outlines the planned development milestones for UBM.
Please note: priorities may change depending on community feedback and technical feasibility.

## ✅ Completed
* [x] Project setup (Forge 1.12.2 base)
* [x] Basic infrastructure for new blocks/items
* [x] Initial GitHub repository
* [x] Backport: **1.15 blocks, items & mobs**
* [x] Backport: **World type "Single Biome"**

## 🚧 In Progress
* [ ] Backport: **1.17 blocks & items**
* [ ] Backport: **1.20 blocks & items**
* [ ] Improve compatibility with major 1.12.2 mods
* [ ] Performance optimizations
* [ ] Documentation

## 📅 Planned Features
* [ ] Backport: **1.14 mechanics**
* [ ] Backport: **1.16 blocks & items**
* [ ] Backport: **1.18-1.21 blocks & items**
* [ ] Backport: **1.13 blocks, items, mechanics and entites** (waterlogging, dolphins, etc.)
* [ ] **Config system** to enable/disable features individually
* [ ] Enhanced **multiplayer support** for modded servers
* [ ] Expanded **datapack-style customization**

## 💡 Ideas / Under Discussion
* Optional **new world generation** system (toggleable for compatibility)
* Structures
* Integrations with popular mods (e.g., JEI, Biomes O’ Plenty)

## 📌 How to Contribute
* Share your suggestions via [GitHub Issues](https://github.com/JWeinelt/UltimateBackportMod/issues)
* Vote 👍 on features you care about most
* Help us prioritize by giving feedback in discussions
* Create textures and sounds for the mod!

👉 As this is a modification of the game, we are not allowed to use any assets from Minecraft itself. We need to recreate them with our own style.

✨ *UBM is a community-driven project – your ideas shape the future of backporting!* ✨
